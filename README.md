# 🎮 Hangman Game using Python & Tkinter

## 📌 Overview

This project is an interactive **Hangman Game** built using **Python** and Tkinter. The game features a modern graphical user interface, category-based word selection, score tracking, and dynamic visual feedback for incorrect guesses.

Players must guess the hidden word one letter at a time before running out of attempts.

---

# ✨ Features

* 🎨 Modern GUI using Tkinter
* 📂 Multiple word categories
* 🎯 Random word generation
* 🧠 Letter guessing system
* 📊 Score tracking
* ❤️ Visual life/attempt system
* ⚠️ Input validation
* 🏆 Win/Lose pop-up messages

# 🎮 Gameplay

1. Select a category
2. Click **Start Game**
3. Enter one letter at a time
4. Correct guesses reveal letters
5. Wrong guesses reduce attempts
6. Guess the full word before attempts run out
