# 🎮 Console Hangman Game in Python

## 📌 Overview

This project is a console-based Hangman Game developed using Python.
The player must guess a hidden word one letter at a time before running out of attempts.

The game includes multiple categories, score tracking, input validation, and ASCII-based hangman visualization displayed directly in the terminal.

---

# ✨ Features

* 📂 Multiple word categories
* 🎯 Random word generation
* 🧠 Letter guessing system
* 📊 Score tracking
* ⚠️ Input validation
* 💻 Fully console-based gameplay
* 🎮 ASCII Hangman visualization

# 🎮 Gameplay

1. Choose a category
2. Guess one letter at a time
3. Correct guesses reveal letters
4. Incorrect guesses reduce attempts
5. Guess the word before the hangman is completed
